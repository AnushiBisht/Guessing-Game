# 🦀 Guessing Game

> My first steps into the world of Rust — and the beginning of my journey to becoming a **Rustacean**.

## About This Project

This is the classic number guessing game from *The Rust Programming Language* book (aka "the Book"). It's a rite of passage for every beginner learning Rust — my very first taste of the language, straight from Chapter 2.

The program:
- Generates a random secret number between 1 and 100
- Asks the player to guess it
- Tells them if their guess is too high, too low, or just right
- Keeps going until they get it right

Simple on the surface, but it introduces some of Rust's core ideas: variables, mutability, `Result` and error handling, crates, and pattern matching — all wrapped into one small, satisfying project.

## Why I'm Doing This

I'm starting my journey to become a **Rustacean** 🦀 — someone comfortable with Rust's mindset around safety, performance, and ownership. Everyone starts somewhere, and this little guessing game is my "Hello, World" moment for a language I'm excited to actually understand, not just use.

## Running the Game

Make sure you have [Rust and Cargo installed](https://www.rust-lang.org/tools/install), then:

```bash
git clone https://github.com/yourusername/guessing_game.git
cd guessing_game
cargo run
```

You'll be prompted to guess a number. Keep guessing until you find it!
